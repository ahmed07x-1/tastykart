# 🍔 TastyKart – Food Ordering Web App

TastyKart is a responsive food ordering web application developed using HTML, CSS, JavaScript, Axios, and JSON Server. Users can browse food items, add products to cart, and place orders through an interactive UI.

## 🚀 Live Demo
🔗 Website: https://tastykart.netlify.app/

## 📂 GitHub Repository
🔗 Repository: https://github.com/ahmed07x-1/tastykart

## ✨ Features
- Responsive food ordering UI
- Dynamic menu fetching using Axios
- Add to cart functionality
- Place orders
- Backend integration using JSON Server
- Mobile-friendly design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Axios
- JSON Server
